How it works?

Before going on to the anecdotes here is how this works.  One might wonder how these sites can know which sites have been visited.

When a site gets hacked the hacker who is behind this hack sells or releases the site's password database.  Haveibeenpwned uses the logic of antivirus by adding hacker databases  in their own database to be able to cover more site and security.
