In the beginning of this blog I made a post talking about “have you been hacked” and I posted this:

Tell me what you have in the comments.
PS: Comments are certified not Disqus ^^ so not hacked.

But the problem is that I took 2 weeks before being decided on what I will use. Disqus is the easiest choice.

So, I thought about using “Isso” but I don't want to create a full setup to be able to post some comments.

The best option I found is Commento.

What is Commento?

Commento is an easy lightweight privacy-focused alternative to Disqus for a faster web and discussions.

You can install Commento by yourself or using their hosting service “commento.io”. I used the easiest option because I'm too lazy.

How to add Commento to your website

This is a quick tutorial to use the online service of Commento:

  1. Go on https://commento.io/
  2. Sign up
  3. Add a new domain with your website
  4. Copy the code snippet
  5. Paste the code snippet into an HTML box where you want on your blog. I personally used at the end of my posts because if I put it into my footer, this doesn't look good.

Manage messages

To manage the comments just go on the right page and click on V to accept a message or on the rubbish bin to delete the comment. You can also change these settings on the Commento dashboard.

The statistics

You can get statistics about your website through Commento. It is very useful for me because GitHub Pages doesn't have a stat tracker integrated, so I don't know how many people are visiting my blog.

Now I can get nice graphs! Graphs are cools!

Conclusion

If you are looking for a Disqus alternative, stop searching this is Commento. You can get Commento in touch by going to their official website.
