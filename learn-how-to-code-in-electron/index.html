Learn web dev and desktop dev in once

Learn web dev and desktop dev in once

If you want to create a desktop application and/or a web application I recommend you to learn Electron.

The Electron framework works like a website, with HTML, CSS and JS excepted that it is built to be cross-platform and offline desktop applications.

Let's start with the basics...

First, you will need to install Electron on your computer, I also recommend you to install Atom as text editor (By the way, Atom works with Electron, like Discord). Then, I recommend you to install the "vk-terminal" package on Atom to get a built-in terminal.

  1. So let's start by creating a folder for your app, then open a terminal inside the folder.
  2. Run the following command, and answer to the questions:
npm init

3. Run the following command to start Atom in the current directory:

atom .

4. In Atom, open the package.json file and change from test to start. Also, just after licence, adds the following code: 

  "devDependencies": {
    "electron": "^3.0.2"
  }

You should get something like this at the end:

{
  "name": "AIP-GUI",
  "version": "1.0.0",
  "description": "Install any software without terminal!",
  "repository": "",
  "main": "app.js",
  "scripts": {
    "start": "electron ."
  },
  "license": "ISC",
  "devDependencies": {
    "electron": "^3.0.2"
  }
}

5. Create a new file called app.js this app will generate the window and start the app, this is the file that is described in package.json. Paste the following content in it:

const {app, BrowserWindow} = require('electron')
const path = require('path')
const url = require('url')

let window = null

// Wait until the app is ready
app.once('ready', () => {
  window = new BrowserWindow({
    width: 800,
    height: 600,
    backgroundColor: "#D6D8DC",
    show: false
  })

  // Load a URL in the window to the local index.html path
  window.loadURL(url.format({
    pathname: path.join(__dirname, 'index.html'),
    protocol: 'file:',
    slashes: true
  }))

  // Show window when page is ready
  window.once('ready-to-show', () => {
    window.show()
  })
})

6. Create another file called index.html, you can keep it empty for the moment.

7. Open your terminal and run the following commands to install Electron, the NodeJS stuff and start your application:

npm install --save-dev electron
npm install
npm start

At the end an empty window should pop-up, this is your app! To run your app you just need to run the last command you've typed.

Create the interface.

The interface (basic, without styling) will be created using the HTML file, in Atom you can add the basic HTML by writing "html" then TAB key. Also, I recommend you to instal the Atom's package "html-preview" and press CTRL+SHIFT+H.

  1. Add the basic HTML stuff.
  2. Link with the (non-existing) CSS style sheet (in the header):
<link rel="stylesheet" type="text/css" href="./assets/style.css">

3. Add the future script file at the end of "body": Remember that if you need to run function without trigger, you should run them after this line.

<script src="index.js"></script>

4. Add your text and different HTML elements, we will do a very basic thing, an input field, a button and a text field.

<input type="text" name="" value="">
<button type="submit" name="submit">Submit</button>

<p></p>

5. Then, add IDs to make the elements to be located by the code.

<input id="textfield" type="text" name="" value="">
<button id="btn" type="submit" name="submit">Submit</button>

<p id="textzone"></p>

6. Finally, add functions when a specific element is clicked, these functions will call the JS code.

<input id="textfield" type="text" name="" value="">
<button id="btn" onclick="code_submit()" type="submit" name="submit">Submit</button>

<p id="textzone"></p>

5. Then, let's translate this into JS:

function code_submit(e){
  //Get elements by IDs
  var textfield = document.getElementById('textfield');
  var textzone = document.getElementById('textzone');
  var btn = document.getElementById('btn');

  //Take the value of textfield
  var text = textfield.value;
  //Set the value of textzone into textfield
  var display = textzone.innerHTML=text;
  //Return the result
  return false;
}

6. Your app works! You can run it by running:

npm start
