How to make internet browsing faster

To make your usual tasks on internet faster, I already wrote a post about it. Use Brave instead of Chrome.

It removes the ads and make internet much faster than it was before! Just watch this little demo video:

Restrict the startup applications.

You can restrict the startup application who often work in background.

  1. Open the "Startup Application" software
  2. Remove the apps you don't need.
  3. For the other apps, create a sleep time by clicking on edit and writing "sleep 10;" before the command
  4. Save it

How to reduce overheating

Overheating is a common problem with computers (mine is a laptop), it makes everything slower. The first thing maybe not directly linked to overheating problem is: try to plug your AC.

When my battery is low, my computer is really buggy. I don't know if it is true or if it is just a feeling but when I plug my AC it feels better.

Lighter alternatives to Gnome

Gnome is the desktop environment of Ubuntu. If you still have big issues and you need a faster computer right now, then try to install another environment like "XFCE" or "LXDE".

If you find another good alternative you can also install it. You don't need to reinstall another OS to apply it.

If you have more time, I recommend you to check a specialist to check your computer components. Sometimes you will need LXDE or XFCE sometimes, just changing few old components can solve the issue.

The driver issues

Another common issue is the driver issues. I don't have driver issues but if you have a powerful computer you will maybe need a driver to make your computer to run properly.

You can find easily documentation about this issue because many peoples have it. You can also directly check on the official website of your graphic card to find the drivers and the doc.
