How to install a fully anonymous OS

In one of my precedent posts I've talked you about "how to be anonymous on the internet".

I forgot to talk you about Tail, Tail is a portable OS that makes you able to get a fully anonymous OS anywhere. This OS is Tails, and we will see how to install it now!

Requirements

To begin with the installation of Tails, you need few requirements:

  • An empty flash drive
  • The open-source Etcher software, link here.
  • The Tail image, can be downloaded in about 5 minutes.

Flash the drive.

Now, once you've got the requirements follow these steps:

  1. Plug the flash drive into your computer and make a backup.
  2. Open the Etcher software (on Linux, just extract and click on the file, no setup required)
  3. Select the Tails image, select your flash drive and click on Flash.
  4. You just need to wait few minutes. Just drink your tea...
  5. If you did it on another computer, you can take it off.

How to access it?

Yeah, the full installation is finished, so now how can you access it?

  1. Power off the computer you want to use to access to Tails.
  2. Plug the Tails flash drive.
  3. Boot the computer and access to the BIOS, this is the trickiest step.because this depends of the computer:
    Some needs a needle to put into a little hole on the side of the computer but I think most of the computers uses the "F12" key. So spam the F12 key when the computer is starting.
  4. Access to the "boot menu" and select the Tails flash drive. If you got a "Boot Secure error" you can disable it into advanced option > security > Secure Boot the save it and try again.
  5. Finally, just complete the setup of the settings (like the keyboard, etc) Also add all the additional settings.
  6. BONUS: You can also add a "permanent" folder by opening the "Permanent Volume configuration" app and restarting Tails, when you will boot Tails again you will be able to decrypt the volume.

Wi-Fi's issue.

Then, if you cannot connect yourself to Wi-Fi and this guide is not useful to you then I have bad news for you, you need a Wi-Fi dongle to your computer or connect it via Ethernet. Unfortunately mine don't have an Ethernet port and I don't have a Wi-Fi dongle, so I used another computer.

How to use Tails.

You can use Tails using many built-in apps or by installing new apps (apps will be stored on your encrypted permanent volume).

The major built-in apps are:

  • Tor Browser
  • Onion Share
  • All the basic Linux apps (Libre Office, etc)
  • Unsafe browser (Firefox)
  • And many others that I still need to explore, so go check it yourself!
WARNING! Every file in your classic home directory are not stored on Tails (yeah, that's the point.) so if you want to store these files copy them into the "Persistent" directory.

Conclusion

Finally, if you are curious about how Snowden sent the secret docs from the NSA to the journalists, then try Tails and Tor Browser.

So in a nutshell, to start Tails:

  1. Spam the F12 key when boot and select your flash drive
  2. Press ENTER to confirm and login your persistent volume
  3. Add all the additional settings and click on start
  4. Explore and use it!
  5. Shutdown the computer, everything that is not stored in the "persistent" directory excepted software are not stored on the flash drive and are forgotten, for ever.

I hope you enjoyed it and tell me in the comments below what you think about this Debian Distro!
