Hello!

This is a tutorial to teach you how to dockerize an application. So let's get into it!

Requirements

You need few things before starting this tutorial:

  • Docker installed
  • An application built from source with commands, to get started use this one: https://github.com/prakhar1989/docker-curriculum.git

Test the Docker installation

To test if Docker works properly, run the following command :

docker run hello-world

You should have a text displaying saying “hello-world”

Dockerize locally an app

Using the Git I've shown to you above (git repo) you can run look to the file called “Dockerfile” into “flask-app”

To do so, you need 3 basic components:

  • FROM python:3-onbuild to install Python and Flask
  • EXPOSE 5000 to show the port 5000 from the app
  • CMD ["python", ".app.py"] to run the Python file

Then we need to build the image and start it: 

sudo docker build -t catnip .
sudo docker run -p 8888:5000 catnip

The -p argument is used to change from port 5000 to port 8888, the dot after catnip is used to get the current directory.

Now you can go on http://localhost:8888/ to see a cat GIF, the Flask app is running!

Save the image.

Now lets backup the image and send it into Docker Hub.

Then when it's downloaded locally you can install them like this:

sudo docker load < catnip.tar.gz
sudo docker run -p 8888:5000 catnip

Publish the image

To publish the image you need to create a new one with your username:

sudo docker build -t USERNAME/catnip .
sudo docker login
sudo docker push USERNAME/catnip

Then, to run catnip you just need to run the following command:

sudo docker run -p 8888:5000 USERNAME/catnip

Also, If you need more settings you can put them into a docker-compose file and run "docker-compose up".
