How to get mods on Minecraft

by SnowCode

2019-11-14

If you play Minecraft and you want  to install mods like Shaders, additional blocks, new functions, etc then read this post!

How to choose your mod

To choose your mod you can browse YouTube, many YouTubers have tried some mods on their channel.

If you know exactly what purpose or the mod you want you can quickly google it and download it.

If you got more than 1 mod, you need to be sure that all mods have the same Minecraft version.

Also, start the classic version of the mod to create a new version folder into the game folder.

Mod vs Plugin

There are two types of "Mods" there are the mods and the plugins. Make sure the mod(s) you want to install is not a plugin. Plugins can only be installed when you own a Minecraft Server

Once you are sure about that we can start by installing Minecraft Forge.

Installing Minecraft Forge

Forge is the tool to install mods on Minecraft, you need to install it to your Minecraft folder.

  1. Start the classic version of the mod to create a new version folder into the game folder.
  2. Go on the official MCForge website: http://files.minecraftforge.net/
  3. Choose the version that works with your mod (like 1.13 for example) on the side bar on the left
  4. Then into "recommended" click on installer (for Linux and macOS) or Win (for windows), you will be redirected on another page, click on Skip after 5 seconds on the top right of the page.
  5. Open the downloaded file, Some people have issues with dependencies, so if you had some dependency problem try to google the dependency error, install the linked software and restart the installer. If you use Linux you need to go into properties and select "allow executing file as program".
  6. You need to make sure the path to your Minecraft directory is correct, if you used the official launcher it should be true. If not, double check if the path is the same as the official one. Also, check if "Install Client" is chosen.
  7. Click on install and wait a minute.
  8. Start the launcher on the Forge version.
  9. Open the Minecraft folder it is the path you had during the installation. To get more information about the path to the folder: https://help.mojang.com/customer/portal/articles/1480874-where-are-minecraft-files-stored-
  10. In this directory, go into the "mods" folder and paste all your mods in the directory. Check if all mods have the same Minecraft version
  11. Check if your launcher had a version called "Forge-v13.0" for example. Select it and launch Minecraft.
  12. Minecraft is opened, the layout of the main page have changed. A new button called "Mods" appears. This button needs to prove that you are using Forge and to get more information about the installed mods. Enjoy!

Troubleshooting

This is a little section to explain to you the main causes of errors.

Cannot start Minecraft

If you cannot start Minecraft, then clean your mod folder to check if Minecraft can just starts Forge without mods. If it works then check what mod is causing the error.

If you still have the error, then check your Java version. I had Java 11 but I needed Java 8. This is a quick tutorial to solve this issue:

  1. Install Forge
  2. Go into the "forge" profile in the launcher and go into "more options"
  3. Into Java executable write "/usr/lib/jvm/java-8-openjdk-amd64/bin/java" to find this path I ran "update-alternatives --list java" and added "/bin/java"  to get the file
  4. 4. Save and run Minecraft

NB: This has been made on a Forge and Minecraft installation. If you still get issues try to clean the mod folder to find which mod cause the problem.

Minecraft don't want to launch (even in classic version)

If you are at the first step of this tutorial and you are stuck then follow these steps:

  1. Open your launcher and click on edit profile
  2. Into JVM arguments remove " -XX:+CMSIncrementalMode"

The launcher doesn't want to load Forge in the version list.

Then the problem comes from the launcher try to update it, if it's not  This problem cannot occur when using the official launcher. You probably use Shiginima Launcher.

To solve this issue do the following steps:

  1. Go into the game directory
  2. Into "versions" open the forge folder
  3. Open the file
  4. Under "inheritsFrom" write the following line: (by changing the version)
    "jar": "1.14.4",
  5. Then, into libraries change "url": "" into "url": "http://files.minecraftforge.net/maven/"

Save and launch Minecraft

NB: I found this solution into Spanish video: https://www.youtube.com/watch?v=oy6aau8HfWw
