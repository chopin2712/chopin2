Amazing open source password manager

I tested some passwords managers and I observed that there is multiple managers types.

You got the file based, like an encrypted table sheet. The problem is that is not sync with other devices. Like Keepass

Another type is the most popular is the online registration based manager like Lastpass for example. The problem is that many of them are getting hacked and all your passwords pwned. So I don't recommend this option.

So the last type is unique, I only had observed is called "Master". It works with cryptography and encryption of your passwords.

How it works

Master works with 4 data, the username, the password, the website name, the encryption settings (like long password, short password, maximum, pin, etc)

So it creates a hash with all of these data when you enter the website URL for example.

So by this way it is sync to all devices on earth by the power of mathematics!

How to install

You can install Master on every platforms, even just online!

This is the link to the official website: https://masterpassword.app/

Give it a try on this website: https://js.masterpassword.app/

How to use

You can use it very easly

  1. Login and create an account at the same time by typing your real name, the master password (the one you will remember)
  2. Type the website's URL like "twitter.com"
  3. Copy and paste the calculated password to login and close the window.

Negative point

One thing I wanted to have with this is an addon and plugin so you will just have to right click and click on login, then right click and click on password, give your master password and then click on login.

Because on all version the password is shown on clear on the screen which is weird for a password. Immagine write "paypal.com" and your password is wrote just above to any peoples who are near to your screen.

Another thing is that the mobile app is not setup like the desktop and online one, it is set to "Maximum Security" setting by default but on desktop and online it is set on "Long Password" settings, so when you add a password on your phone, do not forget to change the settings.

Get the source

This is an open source software you can get the source over there: https://github.com/Lyndir/MasterPassword

Conclusion

This password manager is sync by maths and very easy to use. protected by maths and not known. So hackers will probably not think about Master as password manager, cannot hack it or hack database because no data is stored.

If they pwned you on a website you are protected, you just needs to change the password you have on this website, all other websites are safe because they don't have the same password!

I highly recommends you to test it with the online app.
