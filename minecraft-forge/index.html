If you play Minecraft and you want  to install mods like Shaders, additional blocks, new functions, etc then read this post!

You need to choose your mod before starting this guide!

How to choose your mod

To choose your mod you can browse YouTube, many YouTubers have tried some mods on their channel.

If you know exactly what purpose or the mod you want you can quickly google it and download it.

If you got more than 1 mod, you need to be sure that all mods have the same Minecraft version.

Mod vs Plugin

There are two types of "Mods" there are the mods and the plugins. Make sure the mod(s) you want to install is not a plugin. Plugins can only be installed when you own a Minecraft Server

Once you are sure about that we can start by installing Minecraft Forge.

Installing Minecraft Forge

Forge is the tool to install mods on Minecraft, you need to install it to your Minecraft folder.

  1. Go on the official MCForge website: http://files.minecraftforge.net/
  2. Choose the version that works with your mod (like 1.13 for example) on the side bar on the left
  3. Then into "recommended" click on installer (for Linux and macOS) or Win (for windows), you will be redirected on another page, click on Skip after 5 seconds on the top right of the page.
  4. Open the downloaded file, Some people have issues with dependencies, so if you had some dependency problem try to google the dependency error, install the linked software and restart the installer. If you use Linux you need to go into properties and select "allow executing file as program".
  5. You need to make sure the path to your Minecraft directory is correct, if you used the official launcher it should be true. If not, double check if the path is the same as the official one. Also, check if "Install Client" is chosen.
  6. Click on install and wait a minute.
  7. Open the Minecraft folder it is the path you had during the installation. To get more information about the path to the folder: https://help.mojang.com/customer/portal/articles/1480874-where-are-minecraft-files-stored-
  8. In this directory, go into the "mods" folder and paste all your mods in the directory. Check if all mods have the same Minecraft version
  9. Check if your launcher had a version called "Forge-v13.0" for example. Select it and launch Minecraft.
  10. Minecraft is opened, the layout of the main page have changed. A new button called "Mods" appears. This button needs to prove that you are using Forge and to get more information about the installed mods. Enjoy!

Troubleshooting

This is a little section to explain to you the main causes of errors.

Minecraft crashes when going into a world (shaders)

If you use shaders or other things to improve your graphical experience then you need to have a good PC and a good configuration. You need to double-check if your computer is powerful enough to run this launcher.

There is often a "configuration" section when you download a mod.

Cannot start Minecraft

If you cannot start Minecraft, then clean your mod folder to check if Minecraft can just starts Forge without mods. If it works then check what mod is causing the error.

If it doesn't work, then search support online, this can be a problem with the version itself. Check if other versions of your mods are also corrupted. For example, I cannot install Forge 1.14.4 but I can install 1.12.2.

The launcher doesn't want to load Forge (cannot start MC too)

Then the problem comes from the launcher try to update it, if it's not  This problem cannot occur when using the official launcher. Then change the launcher. This problem cannot occur when using the official launcher.

Also, if you need help with Forge and your launcher, you will not find any help on the Forge Forum. The forge forum is very hostile to crack launcher users.

So if you had an issue like this one, bad luck. I recommend you to make a backup of your precedent installation and install the official launcher and finally Minecraft Forge.

Cannot install Forge

If you simply can't install Forge, check your path and your dependencies to make sure it is correct. If it is correct try another version or search support or bug reports on internet. I think it is not a very common issue.

Conclusion

Now you can install many mods on Minecraft, if you are interested in plugins then read my post about How to create a Minecraft Server. Also, if you need help you can write a comment or search on the internet for similar issues.

Thanks for reading this post, I hope it helped you!
